Calculadora inteligente
Este es el programa de una calculadora que pedira que operacion hacer o si cerrar el programa

Al principio se le pedira al usuario que quiere hacer, una suma, resta, multiplicación o division.
![image](https://github.com/user-attachments/assets/3304e0b6-9570-4696-8b6c-9c536b59526e)

Al ingresar un número se le pedira al usuario un número
![image](https://github.com/user-attachments/assets/9543d395-392e-446a-b46d-af430aff24c2)

Despues se le pedira otro número y se le dara el resultado
